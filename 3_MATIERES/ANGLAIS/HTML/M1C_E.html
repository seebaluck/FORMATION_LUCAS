M1C — Classroom English

E = explication. Objectif : comprendre les consignes essentielles du professeur en anglais.

1) Objectif du module

  • comprendre les consignes courantes en classe ;
  • réagir rapidement quand le professeur parle ;
  • reconnaître les verbes d’action : open, close, listen, repeat, read, write, look ;
  • se préparer aux consignes du collège.

2) Les consignes essentielles

3) Règle simple

Beaucoup de consignes commencent directement par un verbe d’action.

Open... = ouvre...

Close... = ferme...

Listen = écoute

Repeat = répète

4) Pièges à éviter

Book = livre.

Notebook = cahier.

Look = regarde, mais book = livre.

Write commence par un W muet : on prononce plutôt raït.

5) Exemples de classe

Teacher: Open your book, please.
Lucas: Lucas ouvre son livre.
Teacher: Listen and repeat.
Lucas: Lucas écoute puis répète.
Teacher: Read the sentence.
Lucas: Lucas lit la phrase.
Teacher: Work in pairs.
Lucas: Lucas travaille avec un camarade.

7) Vocabulaire scolaire

Instruction = consigne Teacher = professeur Student = élève Classroom = salle de classe Work in pairs = travaillez par deux

8) À retenir

Open / Close : ouvrir / fermer.

Listen / Repeat : écouter / répéter.

Read / Write : lire / écrire.

Stand up / Sit down : se lever / s’asseoir.

Work in pairs : travailler par deux.
